
  Данный каталог содержит тексты пакета подпрограмм, реализующих
  примитивы работы с экраном и клавиатурой в ОС MSDOS и ОС Unix,
  а также осуществляющих поддержку создания системы меню,
  диалоговых окон и пр.

  Файл		Описание
  ────────────────────────────────────────────────────────
  SCRDOS.CC	- нижний уровень пакета для ОС MSDOS
  SCREEN.CC	- нижний уровень пакета для ОС Unix
  TTY.CC	- стыковка с драйвером терминала в ОС Unix
  KEY.CC	- работа с клавиатурой терминала в ОС Unix
  MENU.CC	- реализация системы меню
  POPUP.CC	- реализация окон запроса
  DIALOG.CC	- реализация диалоговых окон
  BOX.CC	- поддержка диалоговых окон
  TERMCAP.CC	- работа с базой данных терминалов (termcap) в ОС Unix
  TERMLIB.CC	- стыковка со стандартной библиотекой termlib в ОС Unix
  TERMGOTO.CC	- разбор строки адресации курсора (для termcap)
  CURSES.CC	- поддержка совместимости с библиотекой curses
  HV.CC		- пример законченной программы с использованием данного пакета
  HELLO.CC	- пример простейшей программы
  HELLCURS.C	- пример, демонстрирующий совместимость с биботекой curses
  CTYPE.C	- вспомогательный файл для сквозной руссификации
  SCREEN.H	- основной файл определений
  MENU.H	- определения для подсистемы меню
  POPUP.H	- определения для подсистемы окон запроса
  DIALOG.H	- определения для подсистемы диалоговых окон
  TERMCAP.H	- определения для подсистемы termcap
  EXTERN.H	- общие определения
  EXTDOS.H	- определения для совместимости с MSDOS
  CURSES.H	- определения подсистемы совместимости с curses
  CTYPE.H	- вспомогательный файл определений для сквозной руссификации
  MAKEFILE	- управляющий файл для сборки библиотеки
  MAKEFILE.UNX	- управляющий файл для сборки библиотеки в ОС Unix
  CURSES-R.TXT	- описание функций совместимости с библиотекой curses
  README-R	- этот файл
  README	- то же самое по-английски
